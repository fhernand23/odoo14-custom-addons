# Argentina VAT Ledger

Creates Sale and Purchase VAT report menus in "Accounting/Reporting/Argentina Statements"

## Digital Aliquots (IVA Digital)

For export into Digital Aliquots format (TXT file) first you need create a VAT Ledger with the period (like the current month). Next you need go to "IVA Digital" tab, press the "Generar Informaci√≥n" button and and you can download the TXT.
